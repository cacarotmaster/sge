# Sistema de Gestión Escolar - IE Rafael Uribe Uribe

Repositorio del Sistema de Gestión Escolar de la Institución Educativa Rafael Uribe Uribe de Medellín.

## Gestión Misional

Este sistema cubre las 3 gestiones misionales de la escuela:

- **Gestión Académica**: Material educativo, planes de área, recursos didácticos
- **Gestión Comunitaria**: Proyectos comunitarios, participación ciudadana
- **Gestión Directiva**: Documentos administrativos, planes institucionales

## Estructura

```
sge/
├── gestion-academica/    # Material educativo por área
├── gestion-comunitaria/  # Proyectos comunitarios
├── gestion-directiva/    # Documentos administrativos
├── assets/              # Recursos compartidos (CSS, imágenes)
└── docs/                # Documentación del proyecto
```
